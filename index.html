DIGITAL AUTHENTICATION

INTEGRATED

SECURITY

Veritech's Digital Authentication is a user-friendly POS authentication system in which physical security products such as holograms & tamper seals are integrated with an additional layer of digital security.

It is a robust, multi-layered solution which enables the end consumer to validate the authenticity of any product by scanning a UID Code (Unique Identification) with our in-house developed mobile application.

HOW IT

WORKS

Veritech Plus uses a proprietary algorithm to generate UID codes for each product and packaging unit. Codes will be embedded on physical media such as seals or labels and can be hidden in different ways such as scratch panels or other covert methods.

To verify if a product is genuine, a consumer can scratch off this panel to reveal the hidden code and simply scan the proprietary QR code via the Veritech Plus app. Alternatively, the product can also be verified on the Veritech Plus website using the UID code.
